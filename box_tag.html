Box tag

10 or more people, timer

1. Set the timer for 10 seconds
2. Define the edges of an area where people must stay inside.
3. Everyone goes inside the box and someone is randomly chosen to be in.
4. Start the timer, then the person who is in can run around and tag someone while everyone else runs away.
5. Whoever is in at the end of the 10 seconds is out.
6. Every so often, shrink the box slightly so there is just enough room for everyone still playing.
