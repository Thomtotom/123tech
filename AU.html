About this website

This website is made for people who are bored or have nothing to do. We've written this website in lockdown (clarification: Covid-19 lockdown 2020), that’s part of the reason we built this in fact so that we can spread ideas to people who have nothing else to do and need something fun, creative and something that you can play by yourself or with others. We are competing in a competition called 123tech. 123tech is about young coders and problem-solvers having the opportunity to make an app or a website. If you were to join, you would contribute to solving a problem in your community by using digital technology. There are prizes.

About us

We are based in Wellington, New Zealand, This website was coded by Tom and Theo. The writing was done by the whole team . The ideas were created by everyone. This page was written by Tom who is writing in third person.

How was this website made?

This website was coded mainly using html5, css3, and javascript. We use visual studios as our web editor. We have two people working with visual studios. We use branches to share each other's work with github you can visit us now @Github 123tech.We put this on the web using a piece of software called Surge. Surge is a front end static website publisher Visit it here.If you look slightly up in our url you can see that the domain is surge.sh that is the default domain for surge (if we were to get another it would cost $20-30).

How do I use this website?

This website is about finding the best activity for you. Our website has a series of questions that then narrows the activities that you're looking for. We have many choices but we recommend our tag page because it has more than 17 different tag games! We can also recommend our card game page. If you’re looking for something specific, we have the search bar, which is case sensitive.
